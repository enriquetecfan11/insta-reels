# Configuración (Setup)

Requisitos y pasos para poner en marcha el proyecto.

- **Node.js** — Versión compatible con Remotion (ver `package.json` / documentación de Remotion).
- **Instalación:** `npm install`
- **Studio:** `npm run studio`
- **Render:** `npm run render` o `npm run render-all`

Para más detalle sobre creación de vídeos y comandos, ver [CREAR_VIDEOS.md](../../CREAR_VIDEOS.md) y [docs/index.md](../index.md).
